## 背景

管理员用户详情弹窗加入预约历史后，内容高度明显增加。当前弹窗没有限制最大高度，也没有内部滚动区，导致记录较多时底部内容无法完整看到。

## 目标

- 让用户详情弹窗在桌面和手机上都能滚动到底
- 不改变现有数据逻辑和交互入口
- 保持弹窗头部稳定，重点内容在弹窗内部滚动

## 方案

- `au-modal-profile` 改为纵向弹性布局
- 为弹窗设置 `max-height`，限制在视口范围内
- `au-modal-body` 在用户详情弹窗中开启内部滚动
- 手机端同步收紧高度和内边距，避免底部按钮被挡住

## 验证

- 用户详情中预约历史很多时，仍能滚动到底部
- “加载更多”按钮可见且可点击
- 桌面和移动端都能正常关闭弹窗和查看大图
